# Changelog

Frontend release notes for MusicHoarder live on the GitHub Releases page:

**https://github.com/Jeffreyyvdb/MusicHoarder/releases**

Each release is cut by [semantic-release](https://github.com/semantic-release/semantic-release) from the [Conventional Commits](https://www.conventionalcommits.org/) history on `main` touching `frontend/`. Tags use the form `frontend-v${version}` so they don't collide with any future backend tagging scheme. See `CLAUDE.md` → "Releases" for the prefix → bump mapping.
